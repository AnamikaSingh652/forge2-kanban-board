# Forge 2 Kanban Board

AI-powered Kanban Board built for NMG Forge 2 using OpenClaw and GPT-5.5.

## Features

- Board Management
- Kanban Lists
- Drag & Drop Cards
- Due Dates
- Tags
- Member Assignment
- REST API Backend

## Tech Stack

### Backend
- Laravel
- PHP
- SQLite

### Frontend
- React
- Vite

### AI Development
- OpenClaw
- GPT-5.5
